အက္ခရာများဖြင့်ရွေ့ခြင်း

သတ်မှတ်ထားသော keys များသည် နှိပ်ပါက အရှေ့သို့ရွေ့လျားပြီး Shift ဖြင့်တွဲနှိပ်ပါက ယခင်နေရာများသို့ပြန်လည်ရွေ့မည် :

  • h: heading
  • l: list
  • i: list item
  • t: table
  • k: link
  • n: nonLinked text
  • f: form field
  • u: unvisited link
  • v: visited link
  • e: edit field
  • b: button
  • x: checkbox
  • c: combo box
  • r: radio button
  • q: block quote
  • s: separator
  • m: frame
  • g: graphic
  • d: landmark
  • o: embedded object
  • 1 to 6: headings at levels 1 to 6 respectively
  • a: annotation (comment, editor revision, etc.)
